# Container App PetShop MFE

### Run local
`npm run start`

**Note:**
You need to run the **pshop-users-ui** mfe

### Run with Docker (WIP)
```
docker build . -t pshop-container-ui

docker run -p 3000:80 -d pshop-container-ui
```

### Using

* Webpack + module federation
* antd
* Trunk base development
